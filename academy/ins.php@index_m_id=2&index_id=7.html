活動學習手冊解答版 - 冰淇淋篇

【冰淇淋篇】

 
冰淇淋製作的材料分別有哪些功用呢？
 1. 砂糖 : 增加甜度。2. 優質鮮奶 : 增加風味，讓產品口感更好。3. 鮮奶油: 成份中的脂肪，蛋白質遇冷呈固狀。
        4. 生蛋黃: 含有卵磷脂，是天然乳化劑，可讓全部材料的蛋白質、脂肪、水份產生乳化效果。
 

為什麼冰淇淋吃起來口感會細綿呢？
 鮮奶油脂肪含量高，其中脂肪、蛋白質低溫時會固化。攪拌時注入空氣使脂肪球蓬鬆，產生毛細孔，因此吃起來有軟綿綿的感覺。
 

冰淇淋製作不成功的原因有哪些？
 1. 攪拌速度太慢，注入空氣量不足。2. 冷卻鍋溫度不足。3. 乳脂肪不足。
